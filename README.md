# Transaction Analyzer

Приложение для анализа банковских транзакций. Позволяет анализировать расходы по картам, 
рассчитывать кэшбэк, отслеживать курсы валют и цены акций, а также формировать отчеты.

## Функциональность

### Веб-страницы
- Главная страница с приветствием, информацией по картам, топ-транзакциями, курсами валют и ценами акций

### Сервисы
- **Выгодные категории кэшбэка**: анализ потенциального кэшбэка по категориям
- **Инвесткопилка**: расчет накоплений от округления трат

### Отчеты
- **Траты по категории**: анализ расходов за последние 3 месяца

Приветствие в зависимости от времени суток
- Информацию по банковским картам (потрачено за месяц, кэшбэк)
- Топ-5 транзакций
- Текущие курсы валют (из пользовательских настроек)
- Текущие цены акций S&P500 (из пользовательских настроек)
- Анализ расходов (общая сумма, средний чек, топ категорий)

## Установка

1. Клонируйте репозиторий:

git clone <url-репозитория>
cd finance-views
Установите зависимости через poetry:


poetry install
Создайте файл .env на основе .env_template и укажите API ключи:


cp .env_template .env
# Отредактируйте .env файл, добавьте ваши API ключи
Настройте пользовательские валюты и акции в файле user_settings.json
